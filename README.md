#  Smart Study Planner

Smart Study Planner is a simple and interactive web application designed to help students manage their study schedule effectively.

##  Features
- Add subjects with exam dates
- Prevents duplicate subjects
- Generates daily study plan automatically
- Expandable subject-wise task view
- Mark tasks as completed
- Automatically removes subject after full completion (with user confirmation)
- Data persistence using browser localStorage

##  Technologies Used
- HTML
- CSS
- JavaScript

##  How It Works
Users can add subjects along with exam dates. The system calculates the number of days remaining and generates a structured study plan. Tasks can be marked as completed, and once all tasks are done, the subject is removed after confirmation.

##  Purpose
This project helps students stay consistent and organized by providing a clear and trackable study plan.
